# 假透明去背工具

這是一個本機網頁工具，用來處理「看起來像透明背景，但其實後面是灰白格子」的圖片。

## 使用方式

1. 打開 `index.html`。
2. 把圖片拖進畫面，或點「選擇或拖曳圖片」。
3. 看右邊結果，如果還有格子殘留，就調高「去除強度」。
4. 如果人物或物品邊緣有灰灰的格子痕跡，就調高「邊緣柔化」。
5. 按「下載透明 PNG」。

## 小提醒

- 這個工具不會把圖片上傳到網路，所有處理都在瀏覽器裡完成。
- 如果圖片裡的主體本身也有很接近灰白格子的顏色，可能會被一起去掉。這時可以把「去除強度」調低一點。
- 如果自動偵測不準，可以按「取深格」或「取淺格」，再到原圖上點一下真正的背景格子。

## 新名詞

- PNG：一種圖片格式，可以保留真正透明的背景。
- 透明：不是白色，也不是灰白格子，而是真的沒有背景。
- 容忍度：這裡叫「去除強度」，數字越高，越多接近背景格子的顏色會被去掉。
